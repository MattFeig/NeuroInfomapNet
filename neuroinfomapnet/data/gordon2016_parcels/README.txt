

The gordon_ptseries original order is in the order of the parcel centroids listed in Parcels.xlsx

They are then reorganized based off alphabetical network label (e.g.; reorder_indices = [9,63,64,65,66,67,68 ...])

The network labels are based off Gordon (2016).

You could find the parcel set and associated group average network labels here:
